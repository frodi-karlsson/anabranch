Anabranch

A Deno-first TypeScript monorepo for async utilities.

  • anabranch
    Async stream processing where errors are collected alongside values instead of stopping the pipeline. Built on Task and Channel primitives.
    JSR npm [![bundle](https://deno.bundlejs.com/?q=anabranch&badge=minified&badge-style=flat)](https://bundlejs.com/?q=anabranch)
  • web-client
    Modern HTTP client built on fetch with automatic retries, timeouts, and rate-limit handling. Returns Task for composable error handling.
    JSR npm [![bundle](https://deno.bundlejs.com/?q=@anabranch/web-client&badge=minified&badge-style=flat)](https://bundlejs.com/?q=%40anabranch%2Fweb-client)
  • broken-link-checker
    Crawl websites and find broken links. Uses web-client for robust HTTP and anabranch streams for concurrent processing with backpressure.
    JSR npm [![bundle](https://deno.bundlejs.com/?q=@anabranch/broken-link-checker&badge=minified&badge-style=flat)](https://bundlejs.com/?q=%40anabranch%2Fbroken-link-checker)
  • fs
    Streaming file-system utilities. Read lines, walk directories, glob for files, and watch for changes — all returning Source or Task for composable error handling.
    JSR npm [![bundle](https://deno.bundlejs.com/?q=@anabranch/fs&badge=minified&badge-style=flat)](https://bundlejs.com/?q=%40anabranch%2Ffs)
  • db
    Database primitives with Task/Stream semantics for error-tolerant async operations. Core interfaces and in-memory adapter for testing.
    JSR npm [![bundle](https://deno.bundlejs.com/?q=@anabranch/db&badge=minified&badge-style=flat)](https://bundlejs.com/?q=%40anabranch%2Fdb)
  • db-postgres
    PostgreSQL database connector using node:pg with connection pooling and cursor-based streaming for large result sets.
    JSR npm [![bundle](https://deno.bundlejs.com/?q=@anabranch/db-postgres&badge=minified&badge-style=flat)](https://bundlejs.com/?q=%40anabranch%2Fdb-postgres)
  • db-sqlite
    SQLite database connector using Node.js built-in node:sqlite for in-memory or file-based databases.
    JSR npm [![bundle](https://deno.bundlejs.com/?q=@anabranch/db-sqlite&badge=minified&badge-style=flat)](https://bundlejs.com/?q=%40anabranch%2Fdb-sqlite)
  • db-mysql
    MySQL database connector using mysql2 with connection pooling for MySQL databases.
    JSR npm [![bundle](https://deno.bundlejs.com/?q=@anabranch/db-mysql&badge=minified&badge-style=flat)](https://bundlejs.com/?q=%40anabranch%2Fdb-mysql)
  • queue
    Message queue with Task/Stream semantics for error-tolerant message processing. In-memory adapter with delayed messages, dead letter queues, and visibility timeout.
    JSR npm [![bundle](https://deno.bundlejs.com/?q=@anabranch/queue&badge=minified&badge-style=flat)](https://bundlejs.com/?q=%40anabranch%2Fqueue)
