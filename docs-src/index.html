Anabranch

A Deno-first TypeScript monorepo for async utilities.

  • anabranch
    Async stream processing where errors are collected alongside values instead of stopping the pipeline. Built on Task and Channel primitives.
    JSR npm
  • web-client
    Modern HTTP client built on fetch with automatic retries, timeouts, and rate-limit handling. Returns Task for composable error handling.
    JSR npm
  • broken-link-checker
    Crawl websites and find broken links. Uses web-client for robust HTTP and anabranch streams for concurrent processing with backpressure.
    JSR npm
  • fs
    Streaming file-system utilities. Read lines, walk directories, glob for files, and watch for changes — all returning Source or Task for composable error handling.
    JSR npm
  • db
    Database primitives with Task/Stream semantics for error-tolerant async operations. Core interfaces and in-memory adapter for testing.
    JSR npm
  • db-postgres
    PostgreSQL database connector using node:pg with connection pooling and cursor-based streaming for large result sets.
    JSR npm
  • db-sqlite
    SQLite database connector using Node.js built-in node:sqlite for in-memory or file-based databases.
    JSR npm
  • db-mysql
    MySQL database connector using mysql2 with connection pooling for MySQL databases.
    JSR npm
  • queue
    Message queue with Task/Stream semantics for error-tolerant message processing. In-memory adapter with delayed messages, dead letter queues, and visibility timeout.
    JSR npm
  • queue-redis
    Redis adapter for @anabranch/queue using ioredis. Supports all queue features with Redis streams for persistent messaging.
    JSR npm
  • queue-rabbitmq
    RabbitMQ adapter for @anabranch/queue using amqplib. Supports all queue features with RabbitMQ queues for persistent messaging.
    JSR npm
